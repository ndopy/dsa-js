# 이진 트리 (Binary Tree)

## 1. 개념
각 노드가 최대 두 개의 자식 노드(왼쪽 자식, 오른쪽 자식)를 가지는 계층적 자료구조이다. 모든 노드는 0개, 1개 또는 2개의 자식 노드를 가질 수 있다.

## 2. 핵심 특징
1. **재귀적 구조:** 트리의 각 노드는 그 자체로 서브 트리(Subtree)의 루트 노드가 된다. 따라서 트리 연산은 재귀적으로 구현하기 적합하다.
2. **비선형 자료구조:** 데이터가 일렬로 나열되는 선형 구조(배열, 연결 리스트 등)와 달리, 데이터가 계층적으로 구성된다.
3. **주요 용어:**
   - **루트(Root):** 트리의 최상위 노드
   - **리프(Leaf):** 자식이 없는 말단 노드
   - **서브 트리(Subtree):** 부모 노드와 연결된 하위 트리

## 3. 트리 순회 (Traversal)
트리의 모든 노드를 특정한 순서대로 한 번씩 방문하는 방법이다.
1. **전위 순회 (Pre-order):** 루트(Root) → 왼쪽(Left) → 오른쪽(Right)
   - 트리의 구조를 복사하거나 전송할 때 유용하다.
2. **중위 순회 (In-order):** 왼쪽(Left) → 루트(Root) → 오른쪽(Right)
   - 이진 탐색 트리(BST)에서 데이터를 오름차순으로 정렬하여 가져올 때 사용된다.
3. **후위 순회 (Post-order):** 왼쪽(Left) → 오른쪽(Right) → 루트(Root)
   - 하위 노드부터 처리해야 하는 경우(예: 폴더 용량 계산, 트리 삭제)에 유용하다.

## 4. 구현 팁
### 1. 재귀(Recursion) 활용
- 이진 트리는 재귀적인 구조를 가지므로, 순회 메서드 구현 시 재귀 호출을 사용하면 코드가 간결해진다.
- **기저 사례(Base Case):** 전달받은 노드(`tree`)가 `null`일 경우 재귀를 종료한다.

**[코드 예시: 전위 순회]**
```javascript
preOrderTraversal(tree) {
  if (tree === null) return; // Base Case

  console.log(tree.data); // 1. Root 방문
  this.preOrderTraversal(tree.getLeftSubTree()); // 2. Left 서브 트리 순회
  this.preOrderTraversal(tree.getRightSubTree()); // 3. Right 서브 트리 순회
}
```

### 2. 노드 클래스 설계
- 데이터와 왼쪽/오른쪽 자식 노드에 대한 참조를 멤버 변수로 가진다.
- 리프 노드 확인(`isLeaf`) 같은 편의 메서드를 추가하면 활용도가 높다.
