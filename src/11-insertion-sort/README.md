# 삽입 정렬 (Insertion Sort)

## 1. 개념
배열의 두 번째 원소부터 시작하여, 그 앞(왼쪽)의 **이미 정렬된 부분**과 비교하여 자신의 위치를 찾아 **삽입(Insert)** 하는 방식으로 정렬하는 알고리즘이다. <br/>
손안의 카드를 정렬하는 방법과 유사하며, 정렬된 부분의 적절한 위치에 값을 끼워 넣는다.

## 2. 핵심 특징
1. **단순함:** 구현이 간단하고 직관적이다.
2. **제자리 정렬 (In-place Sorting):** 데이터를 저장하는 배열 외에 추가적인 메모리 공간이 거의 필요 없다.
3. **안정 정렬 (Stable Sort):** 값이 같은 요소들의 상대적인 순서가 유지된다. (선택 정렬과 다른 점)
4. **데이터 상태에 민감:** 데이터가 이미 거의 정렬되어 있는 경우 매우 효율적이다.

## 3. 시간 복잡도 (Time Complexity)
1. **최악, 평균:** O(n²) - 역순으로 정렬되어 있거나 무작위인 경우.
2. **최선:** O(n) - 이미 정렬되어 있는 경우, 내부 루프를 돌지 않고 비교만 수행하므로 매우 빠르다.

## 4. 구현 팁 (성능 최적화)
- **적절한 시프트(Shift):** 값을 매번 교환(Swap)하는 대신, 삽입할 위치를 찾을 때까지 앞의 원소들을 뒤로 한 칸씩 미는(Shift) 방식을 사용하여 연산 비용을 줄인다.
